codex-proxy-rs

Rust 重写版 codex-proxy：将 OpenAI 兼容请求转发到 ChatGPT Codex 后端。

Endpoints

  • GET /health
  • GET /stats
  • POST /refresh
  • POST /check-quota
  • POST /v1/responses
  • POST /v1/chat/completions
  • POST /v1/messages
  • POST /v1/responses/compact
  • GET /v1/models

Quick Test

curl -sS http://127.0.0.1:8080/health

更多用法见仓库内 README.md。
